## Classical ML Models
- **Decision Tree** — splits data into branches to reach decisions
- **Linear Regression** — predicts continuous outcomes
- **Logistic Regression** — converts probabilities into yes/no outcomes using a threshold

## Key Concepts

**Threshold**
A set point that converts a probability into a final yes/no outcome.

**Contrastive Learning**
AI learns by comparing things — what is similar vs different. Used in image and speech recognition.

**Pretext Tasks**
Small challenges a model creates from its own data (e.g. predicting missing words, reconstructing images). Not the final goal but helps build strong internal representations.

**Anomaly Detection**
Finds data points that don't follow the usual pattern — flags mistakes, security issues, or unusual changes.

**Clustering**
Groups similar data points based on characteristics. Finds natural patterns even in unlabeled data.

## Unsupervised Learning
Both clustering and anomaly detection are used in unsupervised ML to make sense of large amounts of unlabeled data.
